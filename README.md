# AI Affiliate Blog

Fully automated AI-powered affiliate website for:
**AI Tools & Productivity for Students and Developers**

## What this does:
- Auto-fetches trending topics
- Generates SEO articles using Gemini / OpenAI / Claude
- Injects affiliate links automatically
- Pushes markdown to GitHub
- Publishes on Netlify
- (Future) Auto-posts to social media

## Tech Stack:
- FastAPI
- Python
- Next.js
- Markdown
- Netlify
- Gemini API

## How automation works:
Bot runs daily and:
1. Fetches trending topic
2. Generates article
3. Injects affiliate links
4. Pushes markdown to GitHub
5. Netlify rebuilds site
6. Article goes live
